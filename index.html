Jigsaw-Land


Description


A relaxing single player 3D puzzle game with procedurally generated puzzles.


Solve puzzle at your own pace, by combining puzzle tiles to compose back together beautiful landscapes.

Replay the same puzzle but with different puzzle tiles to combine together, since each puzzle is procedurally generated.

Use the coins given as a reward for completing a puzzle, to unlock new puzzles and navigate a map with 26 puzzle to solve.

After finally solving the puzzle with a joyous celebration of completion, watch the compose landscape come to life, with sea waves that start moving and wind starting to blow on trees.


All of this gameplay elements made for a captivating gameplay experience.



Features

  • More than one solution, to solve a puzzle you have to fit all the tiles of the puzzle together and every tile follow specific rules on what can be attached to, so there may be more than one solution to piece together the puzzle tiles.

  • Six biomes, each contains a different set of puzzle tiles.

    Relatively: forest, beach, snowy forest, Texas desert, arctic and autumn forest.

  • Hints system, to hint you on how some puzzle pieces could combine together.
  • Special puzzle tiles, that you obtain if you complete some puzzles without too much difficulty, which you can use to unlock paths that lead to more difficult puzzles.
